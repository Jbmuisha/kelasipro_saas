# kelasipro_saas
KelasiPro SaaS est une plateforme de gestion scolaire multi-établissements (multi-tenant) permettant aux écoles de gérer les élèves, enseignants, classes, présences, notes et frais de scolarité de manière sécurisée et centralisée.
avec pythonsocket programming qui permet au parent de communiquer avec le prof et plusiere different functionalité.
